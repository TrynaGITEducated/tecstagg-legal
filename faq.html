❓ Frequently Asked Questions

What is RUBRIX?

RUBRIX is an AI-powered assignment evaluation tool that helps teachers grade assignments by analyzing them against custom rubrics. Upload an assignment and rubric, and RUBRIX provides instant feedback with scores, strengths, and recommendations.

Is RUBRIX free?

Yes! We offer 3 free analyses per day. Upgrade to Pro for unlimited access, batch uploads, PDF reports, and priority processing.

What file types are supported?

We support: TXT, PDF, DOCX, MD, Python (.py), Java (.java), C++ (.cpp), HTML, CSS, JavaScript (.js), and JSON (.json).

How do I get Pro access?

During beta, Pro access is free with an access code. Visit the Upgrade page and enter a valid code. Contact tecstagg@proton.me for a code.

How accurate is the AI grading?

RUBRIX uses Google's Gemini AI, which provides consistent, rubric-aligned feedback. However, we recommend teachers review all AI-generated feedback before returning to students.

Is my data safe?

Yes. We do not store your uploaded files permanently. Files are processed temporarily for analysis and then discarded. See our Privacy Policy for details.

Can I use RUBRIX for batch grading?

Yes! Pro users can upload up to 50 assignments at once. RUBRIX will process them all and provide a dashboard with scores, grade distribution, and CSV export.

What if I need help?

Email us at tecstagg@proton.me and we'll respond within 24-48 hours.
